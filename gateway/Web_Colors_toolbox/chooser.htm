Three color types

You can choose from 140 color names by clicking on the chart, or you can pick a starting color from the 16 bit palette and adjust it using the buttons marked plus or minus [ 1 ], [ 4 ] and [ 8 ].

Actually, there are only 138 named colors because two of them have alternative names:  Aqua = Cyan,  and  Fuchsia = Magenta.

On the mixer panel the Web 216 equivalent of a named color or a 16 bit color is presented as an alternative choice.  Please note:  it does not necessarily show what the selected color would change to in 256 color mode, because there are 256 possible alternatives, not 216.

How to use the color picker

  • The first selected color is the named color "White". You can change the background color of the page by clicking on color names or on the 16 bit palette. When color names are selected the 16 bit equivalent will be shown also.

  • Alternatively, click on to change the color type. If you choose "16 bit palette" or "Web 216 palette", the color name will be replaced by the words "select named colors". If "Web 216 palette" is selected, the 16 bit values go blank until another color type is selected.

  • Change the text color by clicking on [ Text ].  Not all browsers can change the text color.

  • Switch back to changing the background color by clicking on [ BG ].

  • In 16 bit color mode you can make colors [ lighter ] or [ darker ].

  • Click [ save ] to save a color combination.

  • Click [ recall ] to view the last 10 combinations in a pop-up window.

    The decimal RGB equivalent of the selected color is shown in the top row of buttons.

Web 216 palette

In Internet Explorer for Windows the 256 color palette contains 18 colors which are lighter than CCCCCC, and the 16 bit palette contains more than 600, but the traditional Web 216 palette only has 7 lighter colors. For this reason, the Web 216 equivalent on the mixer panel is set to FFFFFF (white) when there is no close match.  Above CCCCCC, each browser uses different substitute colors in 256 color mode.

The 16 bit palette

Sixteen bit values are divided into Red, Green & Blue components as shown below:

5 bits  -  32 steps  -  Red
6 bits  -  64 steps  -  Green
5 bits  -  32 steps  -  Blue

32 x 64 x 32 = 65536

On Microsoft Windows systems the first increment from zero is a double step. Red & Blue increment in steps of 8, and Green increments in steps of 4 ( hex: 4, 8, C ). The final value is set to FF.

Image colors in 16 bit mode

Image colors can shift in 16 bit mode, resulting in a mismatch with the background. Using transparency in GIF/PNG images is the most flexible solution. Alternatively, if you have image software which can edit the color palette, subtract 1 from the RGB components of the 16 bit color that needs to blend with a 16 bit background color on the page.  Grays in an image aquire a colored tinge unless the RGB values all shift to multiples of 8.

140 color names

When used in HTML tags or CSS, the color names are not case sensitive. There is no systematic naming convention for the color names. For example:

"lightgrey" is spelled with an "e"
"gray" is spelled with an "a"
"lightpink" is darker than "pink"

You can scroll through the list of color names using the [ Page Down ] and [ Page Up ] keys on your keyboard.

Most of the named colors can only be reproduced exactly on a 'True Color' display ( 24/32 bit ).  On a 'High Color' display ( 16 bit ) most browsers will approximate them with a 'dither' pattern, but on a 256 color display ( 8 bit ) the differences can be quite dramatic. It is always a good policy to test web pages in 256 color mode.

CSS 3 : the color names as a standard

The World Wide Web Consortium (www.w3c.org) is the organization which defines official standards for mark-up languages. For web pages, W3C recommends that the information content and the style of presentation - colors, fonts, layouts, etc. - should be marked up separately. The recommended method for defining stylistic elements is to use Cascading Style Sheets (CSS). The list of 140 color names is part of the recommendation for CSS 3, the third level of the CSS specification.
